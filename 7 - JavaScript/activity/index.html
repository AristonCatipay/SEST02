Activity: Library Tracker

Objective: Create an array of book objects. Each book object should have properties like title, author, and isAvailable. Write functions to add new books, display available books, and mark a book as borrowed.

  1. Set up the initial library:
    • Create an array called library.
    • Add three book objects to the array. Each book object should have:
      • title (string)
      • author (string)
      • isAvailable (boolean) to show if the book is currently available
  2. Create functions:

    • Display Available Books: Write a function called displayAvailableBooks that loops through the library array and prints only the books that are available.

    • Borrow Book: Write a function called borrowBook that takes a title as a parameter, searches for the book, and if it is available, changes isAvailable to false and prints a message confirming it was borrowed. If it’s not available, print a message saying it’s already borrowed.

  3. Test the Functions
    • Call displayAvailableBooks to show all available books.
    • Call borrowBook("1984") to attempt borrowing a book.
    • Call displayAvailableBooks again to see the updated list.
